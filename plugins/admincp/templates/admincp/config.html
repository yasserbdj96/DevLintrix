📝 Edit config.json

⚠️ Warning

Be careful when editing this file. Invalid JSON will prevent the application from starting. Always restart the application after making changes.

💡 Configuration Tips

  • APP_NAME: Your application's display name
  • VERSION: Application version number
  • DEBUG: Enable/disable debug mode (true/false)
  • SECRET_KEY: Change this in production for security
  • HOST: Server host address (0.0.0.0 for all interfaces)
  • PORT: Server port number (default: 5000)
  • DATABASE_URI: Database connection string
  • PLUGINS_ENABLED: Array of plugin names to load
